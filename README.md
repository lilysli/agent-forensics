# agent-forensics
An experiment to observe a backdoored coding agent's behaviour in a simulated environment.

Agent: qwen3-max-preview

\notebooks:
- agent_sim.ipynb shows the simulated experiment
- analysis_tool.ipynb shows the tool used to compare the csv files

\src:
- logging.py includes the functions used to do the logging

\results\code: This folder includes the code files generated by the agent for the tasks
\results\logs: This folder includes the logs (.csv) generated during the simulated experiment
